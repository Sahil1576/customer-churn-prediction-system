# 📉 Customer Churn Prediction System

An end-to-end Machine Learning project that predicts whether a customer is likely to churn or not.

## 🚀 Project Overview

Customer churn prediction helps businesses identify customers who are likely to leave their services. This project uses machine learning techniques to predict customer churn and enables proactive customer retention strategies.

---

## 📂 Dataset

- Telco Customer Churn Dataset
- Target Variable: Churn
- Classes:
  - 0 → No Churn
  - 1 → Churn

---

## 🔧 Data Preprocessing

- Missing Value Handling
- Label Encoding
- Standard Scaling
- Train-Test Split
- SMOTE for handling class imbalance

---

## 📊 Exploratory Data Analysis

Performed:

- Correlation Analysis
- Count Plots
- Distribution Plots
- Class Imbalance Analysis

---

## 🤖 Models Used

- Logistic Regression
- K Nearest Neighbors
- Support Vector Classifier
- Gaussian Naive Bayes
- Decision Tree
- Random Forest
- XGBoost

---

## ⚙ Hyperparameter Tuning

RandomizedSearchCV was used to find the optimal parameters.

---

## 🏆 Final Model

XGBoost Classifier
